Navigation Span

You may optionally want to include: 

<span id="navigation">
 <a href="" accesskey="P" 
   onclick="prevSlide(); return false;">Previous</a> |
 <a href="" accesskey="B" 
   onclick="prevSlide(); return false;"></a>
 <a href="" accesskey="N" 
   onclick="nextSlide(); return false;">Next</a>
</span>
in the <head> of each slide.
