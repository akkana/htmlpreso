Navigation Span

Some accesskeys will automatically be added to each slide: 

<span id="navigation">
 <a href="" accesskey="P"
    onclick="prevSlide(); return false;">Previous</a> |
 <a href="" accesskey="B"
    onclick="prevSlide(); return false;"></a>
 <a href="" accesskey="N"
    onclick="nextSlide(); return false;">Next</a>
</span>
You can ignore the nav span entirely when you're giving talks, but it's there if you ever want it.
