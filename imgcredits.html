Image Notes and Credits

Put them in notes.js: 

var notes = {
    "pix/just-image.jpg": "Place note here",
    "pix/img-notes.jpg": "Another note",
}

var credits = {
    "pix/img-notes.jpg": `Image by me,
using GIMP`
}

Don't remove the checkNotes() function.

In JS you can now (finally!) use backquotes for multiline strings as of ECMAScript 6, hurray!

They're supported in these browser versions.
