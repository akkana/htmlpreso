Image Notes and Credits

Put them in notes.js: 

var notes = {
    "pix/just-image.jpg": "Place note here",
    "pix/img-notes.jpg": "Another note",
}

var credits = {
    "pix/img-notes.jpg": "Image by me, using GIMP"
}

Don't remove the checkNotes() function.

This is Javascript code, so the note or credit all has to fit inside the "" as a single string, so you might have something like "Place note here.<p>Here's another paragraph<p>and a third"

I know it's a little unweildy, but that's JS for you.
